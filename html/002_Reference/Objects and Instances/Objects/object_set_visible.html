object_set_visible

Sets the visibility of the given object.

Syntax :

object_set_visible( index, vis );


ArgumentDescription
index The index of the object to change.
vis The new visibility of the object (true=visible, false=invisible).


Returns : N/A


Description

With this function you can flag a specific object as being visible or not (for more information on the "visible" flag see Defining Objects). This means that all instances of this object that are created after visible has been changed will be created with this new state, while instances that are already in the room may not be affected. Please note that this is not an instance function! You can set the visible flag of individual instances and so have ten instances all flagged as visible, but if you change the visible flag of the object to false, all instances that are created of that object will start with visible flagged as false.


Example :

if !object_get_visible(obj_Block) object_set_visible(obj_Block, true);

The above code checks the visible flag of the object "obj_Block" and if it is false then it is changed to be flagged as true.
