object_exists

Returns whether an object with the given index exists or not.

Syntax :

object_exists(obj);


ArgumentDescription
obj The index of the object to check.


Returns : Boolean


Description

This function returns whether an object with the specified index exists or not. Please note, that the value use for checking must have been initialised previously or else you will get an error causing GameMaker:Studio to close.


Example :

if object_exists(obj_temp)
   {
   with (obj_Temp) scr_add_event();
   }

The above code checks the variable "obj_temp" to make sure that it stores the index of an object and then, if it does, it makes the object run a script.
