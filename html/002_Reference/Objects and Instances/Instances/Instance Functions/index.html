Instance Functions

This section deals with functions specificaly for dealing with instances.

Instances are the copies of object resources which have been placed in a room to interact and do things to make your game come to life. The following functions exist that deal with instances.

  1. instance_change
  2. instance_copy
  3. instance_count
  4. instance_create
  5. instance_destroy
  6. instance_exists
  7. instance_find
  8. instance_furthest
  9. instance_id
  10. instance_nearest
  11. instance_number
  12. instance_place
  13. instance_position
