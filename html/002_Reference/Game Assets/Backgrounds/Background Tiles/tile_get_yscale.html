tile_get_yscale

Returns the vertical scaling of a given tile.

Syntax :

tile_get_yscale(index);


ArgumentDescription
index The index of the tile to check.


Returns : Real


Description

Tiles, like instances, can be scaled once created, and this function will return the scaling amount along the vertical axis (where 1 is the default value).


Example :

var tile;
tile = tile_layer_find(-1000, mouse_x, mouse_y);
ys = tile_get_xscale(tile);

The above code will first get the index of a tile at the mouse position with a depth of -1000 and then use that value to store its vertical scale within the variable "ys".
