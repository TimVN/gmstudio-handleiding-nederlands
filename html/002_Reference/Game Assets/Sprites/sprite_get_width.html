sprite_get_width

Finds the width of a sprite.

Syntax :

sprite_get_width(index);


ArgumentDescription
index The index of the sprite to find the width of.


Returns : Real


Description

With this function you can find the width of the base resource sprite, with no transforms, in pixels.


Example :

if sprite_width != sprite_get_width(sprite_index)
   {
   sprite_width = sprite_get_width(sprite_index);
   }

The above code checks the width of the sprite as it is in the current instance and if there is a difference between that and the original base sprite, it resets the width.
