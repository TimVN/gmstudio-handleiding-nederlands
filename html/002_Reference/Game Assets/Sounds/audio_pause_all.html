audio_pause_all

Pauses all sounds.

Syntax :

audio_pause_all();


Returns : N/A


Description

With this function you can pause all sounds that are currently playing.

NOTE : This function will only pause those sounds that have been flagged as "Normal Sound" when loaded into GameMaker:Studio. If the sound is flagged as "Background Music" this function will not work.


Example :

if keyboard_check_pressed(ord("P"))
   {
   global.Pause = !global.Pause;
   if global.Pause
      {
      audio_pause_all();
      }
   else
      {
      audio_resume_all();
      }
   }

The above code checks for a press of the keyboard key "P" and if it detects one it sets the global variable "Pause" to true or false and then either pauses all sounds or restarts all previously paused sounds.
