iap_is_purchased

Indicates whether or not a product has been purchased.

Syntax :

iap_is_purchased(product_id);


ArgumentDescription
product_id The Apple Store/Google Play product id for the purchase in reverse URL format, eg: "com.MacSweeney Games.CTC.LevelPack".


Returns : Boolean


Description

With this function you can find out if the product you are checking has been bought previously or not. If it has then the function returns true and if not then it returns false.


Example :

var purchased;
purchased = iap_is_purchased(string(ds_map_find_value(p_Map, "id")));
if purchased
   {
   draw_text(32, 32, "Purchase : confirmed");
   }
else
   {
   draw_text(32, 32, "Purchase : unconfirmed");
   }

The above code gets the product id from a ds_map and then uses that to test and see if it has been bought or not, displaying a different message depending on the result.
