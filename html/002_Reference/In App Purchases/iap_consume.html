iap_consume

Clears the purchase status of unmanaged purchases.

Syntax :

iap_consume(purchase_id);


ArgumentDescription
purchase_id This can either be the Apple Store/Google Play id string, an URL id, or an index into the available purchases ds_list.


Returns : N/A


Description

This function can be used for clearing the purchase status of any unmanaged purchases (for example, in-game credits). The purchase_id can be either an integer index into the main purchases list, the explicit Apple Store/Google Play id string, or you can pass through the actual identifier name(eg: ".extralevels"), similar to the way that iap_acquire works.


Example :

if mouse_check_button_pressed(mb_left)
   {
   iap_consume(".extralevels")
   }

The above code will clear the purchase status when the mouse button is pressed.
