keyboard_lastkey

Stores the keycode of the last key pressed.

Syntax :

keyboard_lastkey;


Returns : Real


Description

This variable stores the value of the keycode of last key pressed (all standard keycode constants are returned). This variable is not read only and you can change it, for example to set it to -1 if you handled it already.


Example :

if keyboard_lastkey != -1
   {
   str += keyboard_lastchar;
   keyboard_lastkey = -1;
   }

The above code checks to see if the lastkey variable is not equal to -1, and if it is it adds whatever the last key was as a string to the variable "str", then it resets the keyboard_lastkey variable to accept further input.
