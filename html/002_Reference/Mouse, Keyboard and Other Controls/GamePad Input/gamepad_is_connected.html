gamepad_is_connected

Returns whether a given gamepad is currently connected.

Syntax :

gamepad_is_connected(numb);


ArgumentDescription
device Which gamepad "slot" to check.


Returns : Boolean


Description

This function will return whether a gamepad is connected to the given "slot" (returns true) or not (returns false). you would normally use this function in conjunction with the gamepad_get_device_count function to get the correct number of available gamepads and/or gamepad "slots".


Example :

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
   {
   if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
   }

The above code loops throough the available gamepads (or gamepad slots) and then checks each one for a connected gamepad. the returned value is then used to set a global array to true or false for use in future checks.
