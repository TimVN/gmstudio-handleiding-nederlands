device_mouse_y

Returns the y position of the device when pressed.

Syntax :

device_mouse_y(device);


ArgumentDescription
device The device (from 0 - 4) that is being checked.


Returns : Real


Description

This function returns y position of the device. If you are running this on a the HTML5 or PC and Mac modules then this value is updated constantly, as long as the device (usually a mouse) is plugged in, however for mobile devices, this will only be updated while the screen is being touched.

Example :

if device_mouse_check_button(0, mb_left)
   {
   if device_mouse_y(0) > y-32 && device_mouse_y(0) < y+32
      {
      pressed = true;
      }
   else
      {
      pressed = false;
      }
   }

The above code checks to see if the device is being pressed and if so it then polls the device y position to see if it is within the parameters. If it is it sets the variable "pressed" to true, other wise it sets it to false.
