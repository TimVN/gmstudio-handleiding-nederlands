external_free

Frees the DLL with the given name.

Syntax :

external_free(id);


ArgumentDescription
id The name of the dll that you want to free


Returns : N/A


Description

This function frees the memory associated with the dll with the given name. This should be done whenever the dll in question is no longer needed in the game, normally (for example) in an end of game event.


Example :

external_free("MyDLL.dll");

The above example code will free the memory associated with the given dll.
