part_type_color3

Sets a particle type to fade between three colors throughout its lifespan.

Syntax :

part_type_color3( ind, color1, color2, color3 );


ArgumentDescription
ind The index of the particle type to change.
color1 The color the particle will start at.
color2 The color the particle will be halfway through its lifespan.
color3 The color the particle will end at.


Returns : N/A


Description

This function can be used to set a three color gradient for each particle created of the given type. The first color is that which all particles will start with, and the second color is the one that will be blended to half way through it's lifetime and the third color is the one with which the particle will end with. A smooth gradient change will occur through the colors over the particles lifetime from one color to the other.


Example :

part_type_color3( part_Health, c_red, c_white, c_marroon);

The above code will make all particles created of the type indexed in the variable "part_Health" have a color blend from red to white to marroon over their lifetime.
