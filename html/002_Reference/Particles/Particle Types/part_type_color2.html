part_type_color2

Sets the particle type to blend between two colors

Syntax :

part_type_color2(ind, color1, color2);


ArgumentDescription
ind The index of the particle type to change.
color1 The color the particle will start at.
color2 The color the particle will end at.


Returns : N/A


Description

This function can be used to set a two color gradient for each particle created of the given type. The first color is that which all particles will start with, and the second color is the one on which the particle will end with, and a smooth gradient change will occur to the color over the particles lifetime from one color to the other.


Example :

part_type_color2( part_Health, c_red, c_white);

The above code will make all particles created of the type indexed in the variable "part_Health" have a color blend from red to white over their lifetime.
