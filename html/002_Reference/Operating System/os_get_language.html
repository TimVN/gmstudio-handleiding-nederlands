os_get_language

This variable holds a different value depending on the browser or device language.

Syntax :

os_get_language()


Returns : String


Description

This function returns a string with the two letter language code for the OS that is running the game. If the information is not available, it will hold simply an empty string "", or "en" for "English" language.

The following table shows some of the main two letter language codes as defined by ISO 639:

LanguageCode
Arabicar
Chinesezh
Danishda
Englishen
Frenchfr
Germande
Greekel
Italianit
Japaneseja
Norwegianno
Polishpl
Portuguesept
Russianru
Spanishes
Swedishsv



NOTE : This is not the country code that is returned, but the language code of the OS!


Example :

switch (os_get_language())
   {
   case "es": ini_open("spanish.ini"); break;
   case "fr": ini_open("french.ini"); break;
   case "it": ini_open("italian.ini"); break;
   default: ini_open("english.ini"); break;
   }

The above code checks the OS language and opens a different *.ini file depending on the returned value.
