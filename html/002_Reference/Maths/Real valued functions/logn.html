logn

Returns the log base n of val.

Syntax :

logn(n, val);


ArgumentDescription
n The log base.
val The input value.


Returns : Real


Description

This function is similar to the log2(n) and log10(n)functions, only you stiplulate the log base value. For example, logn(5,25) will return how many 5's we need to multiply to get 25 (which is 2).


Example :

logval = logn(5, num);

The above code gets the log5 of the value stored in "num".
