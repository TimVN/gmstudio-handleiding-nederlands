filename_path

Returns the path part of the indicated file name.

Syntax :

filename_path(fname);


ArgumentDescription
fname The file to use.


Returns : String.


Description

Using this function returns the path part of the indicated file name, including the final backslash.

NOTE : This function works on all the C++ target platforms (Windows, Mac, iOS, Android and WinPhone).


Example :

path = filename_path("Test.ini");

The above code gets the path (as a string) of the specified file.
