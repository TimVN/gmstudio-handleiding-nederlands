file_bin_position

Returns the current position (in bytes; 0 is the first position) of the file with the given file id.

Syntax :

file_bin_position(fname);


ArgumentDescription
fname The name of the file to get the position in.


Returns : real.

NOTE : These functions do not work when the target module is HTML5.

Example :

pos = file_bin_position(file);

This would store the current position in the variable "pos".
