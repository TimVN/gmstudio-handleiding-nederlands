file_bin_size

Returns the size (in bytes) of the file with the given file id.

Syntax :

file_bin_size(fname);


ArgumentDescription
fname The name of the file to get the size of.


Returns : real.

NOTE : These functions do not work when the target module is HTML5.

Example :

file = file_bin_open('myfile.bin', 2);
size = file_bin_size(file);
file_bin_close(file);

This would open a file from the same directory as the game, and assign its index to the variable "file". It would then get the size of that file and close it again.
