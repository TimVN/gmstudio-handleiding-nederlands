file_text_eof

Syntax :

file_text_eof(fileid);


ArgumentDescription
fileid The id of the file to check.


Returns : Boolean


Description

This function returns true when the end of a given opened text file has been reached or false if not.


Example :

file = file_text_open_write(working_directory + "\Game_Data.txt");
while (!file_text_eof(file))
   {
   file_text_readln(file);
   }
file_text_write_string(file, level_data);
file_text_close(file);

The above code opens a file for writing then loops through the lines of text already written to the file until it reaches the end. At this point it writes a string and then closes the file again.
