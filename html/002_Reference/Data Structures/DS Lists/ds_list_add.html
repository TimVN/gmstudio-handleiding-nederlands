ds_list_add

Adds the given value at the end of the list.

Syntax :

ds_list_add(id, val);


ArgumentDescription
id The id of the list to add to.
val The value to add to the list.


Returns : N/A


Description

This function can be used to add a new value to the list, and this will be added on at the end of the list.


Example :

ds_list_add(sc_list, score);

the above code will add the value stored in the "score" varriable into the list indexed in the variable "sc_list".
