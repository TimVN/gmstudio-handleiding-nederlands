string_pos

Returns the position of a given sub-string in a string.

Syntax :

string_pos(substr, str);


ArgumentDescription
substr The substring to look for in the string.
str The string.


Returns : String


Description

This function will return the character position of an instance of a substring within a string, or 0 if it's not found at all. One use for this is for filtering words that may be considered offensive, or for finding the correct place to insert some text into another string.


Example :

if string_pos(",", text) != 0
   {
   string_insert(name, text, string_pos(",", text));
   }

The above code searches the string "text" for a comma, and if it finds one it inserts the substring "name" at that position.
