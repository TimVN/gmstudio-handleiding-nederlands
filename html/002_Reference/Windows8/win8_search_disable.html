win8_search_disable

Disable the "search charm" for your game.

Syntax :

win8_search_disable();


Returns : N/A


Description

This function will switch off the ability to search your game for keywords using the "search charm".


Example :

win8_search_disable()

The above code will switch off the search functions for your game.
