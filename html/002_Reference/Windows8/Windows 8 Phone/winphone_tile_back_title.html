winphone_tile_back_title

Set the title for the Flip Live Tile back.

Syntax :

winphone_tile_back_title(title);


ArgumentDescription
title The title of your game tile (a string).


Returns : N/A


Description

With this function you can assign a title string to the back of the Flip type Live Tile associated with your game. This title will only be shown if your tile is classed as being a "flip tile" with a "front" and a "back", in which case this title string will be shown when the tile flips to the "back".


Example :

if os_type = os_winphone
   {
   winphone_tile_title("Catch The Haggis");
   winphone_tile_back_title("MacSweeney Games");
   }

The above code will set both the front and the back title for the games Live Tile to the given strings.
