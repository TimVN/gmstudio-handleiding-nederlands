draw_surface_tiled

Draws a surface tiled so that it fills the entire room.

Syntax :

draw_surface_tiled(index, x, y);


ArgumentDescription
index The index of the surface to draw.
x The x position of where to draw the surface.
y The y position of where to draw the surface.


Returns : N/A


Description

This function will take a surface and then repeatedly tile it across the whole room, starting from the coordinates that you give in the function.

NOTE : When working with surfaces there is the possibility that they can cease to exist at any time due to them being stored in vRam. You should ALWAYS check that a surface exists using surface_exists before referencing them directly. For futher information see Surfaces.


Example :

draw_surface_tiled(surf, x, y);

This will draw the surface indexed in "surf" at the instances own x and y position, and tiled in every direction in the room.
