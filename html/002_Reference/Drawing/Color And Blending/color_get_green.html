color_get_green

Returns the green component of the color given.

Syntax:

color_get_green(col);


ArgumentDescription
col The color to check


Returns : Real


Description

This function returns the amount of green used to make the given color, with the value being between 0 and 255, where 0 is no green and 255 is all green. The following image illustrates this:


Example :

g_comp = color_get_green(c_teal);

The above code will get the green component of the color constant "c_teal" and store the value in the variable "g_comp".
