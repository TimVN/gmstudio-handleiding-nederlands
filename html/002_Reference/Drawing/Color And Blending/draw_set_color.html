draw_set_color

Sets the base color for the draw functions.

Syntax:

draw_set_color(col);


ArgumentDescription
col The color to set for drawing.


Returns : N/A


Description

With this function you can set the base draw color for the game. This value will affect all further drawing where appropriate, including fonts, forms, primitives and 3D. If any of those assets are drawn with their own color value changed, this value will be ignored.


Example :

draw_set_alpha(0.5);
draw_set_color(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_color(c_white);
draw_text(x, y, "LEVEL 1");

The above code will draw some text at the specified position with a shadow effect created by modified draw alpha and color.
