draw_get_color

Gets the current draw color.

Syntax:

draw_get_color()


Returns : Real


Description

This function returns the current draw color which is used for drawing forms, text, primitives and un-textured 3D models. This can be set with the draw_set_color function.


Example :

if draw_get_color() != c_black
   {
   draw_set_color(c_black);
   }

The above code first checks the current draw color and then if it is not set to c_black it sets it to c_black.
