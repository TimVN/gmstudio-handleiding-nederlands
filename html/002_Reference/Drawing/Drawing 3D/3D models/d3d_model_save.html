d3d_model_save

Saves the model to the indicated file name.

Syntax :

d3d_model_save(ind, fname)


ArgumentDescription
ind The index of the model to be saved.
fname The name of the file to save the model to.


Returns : N/A


Description

With this function you can save the indexed model to a text file, which can then be loaded in again using the d3d_model_load() function. You should note that you can actually use almost any format file you like use as long as the name is the same for loading.

NOTE : This functionality is limited to the Standard Version of GameMaker:Studio and above.


Example :

d3d_model_save(model[2], "Terrain_Model.txt")

Saves the model indexed in the variable "model[2]" to a text file.
