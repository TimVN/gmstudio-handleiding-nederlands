d3d_light_define_ambient

Set the color for the ambient lighting of a 3D scene.

Syntax :

d3d_light_define_ambient(color)


ArgumentDescription
color The color of the ambient light.


Returns : N/A


Description

This function is used to control the ambient light of a scene, which is the light that you have in a scene even without having defined any point or directional light sources. It is effectively the overall color and brightness (or darkness) of a scene, and the default color is c_black.


Example :

d3d_light_define_ambient(c_white);

The above code will define the ambient lighting as being white.
