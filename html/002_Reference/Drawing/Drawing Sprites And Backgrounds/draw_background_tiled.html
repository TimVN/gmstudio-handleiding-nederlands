draw_background_tiled

Draws a background at a given position, and tiles it to fill the entire room.

Syntax :

draw_background_tiled(back, x, y);


Argument Description
back The index of the background to draw.
x The x coordinate of where to draw the background.
y The y coordinate of where to draw the background.


Returns : N/A


Description

This function will take a background and then repeatedly tile it across the whole room, starting from the coordinates that you give in the function.


Example :

draw_background_tiled(bck_Sky, x, y);

This will draw the background indexed in "bck_Sky" at the instances own x and y position, and tiled in every direction in the room.
