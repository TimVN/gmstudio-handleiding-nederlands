draw_background_tiled_ext

Draws a background at a given position with scaling, blending and alpha, and tiles it to fill the entire room.

Syntax :

draw_background_tiled_ext(back, x, y, xscale, yscale, color, alpha);


Argument Description
back The index of the background to draw.
x The x coordinate of where to draw the background.
y The y coordinate of where to draw the background.
xscale The horizontal scaling of the background./td>
yscale The vertical scaling of the background.
h The height of the area the stretched background will occupy.
color The color with which to blend the background (use -1 or c_white to display it normally).
alpha The alpha of the background (from 0 to 1 where 0 is transparent and 1 opaque).


Returns : N/A


Description

This function will take a background and then repeatedly tile it across the whole room, starting from the coordinates that you give in the function and with each tile scaled, color blended and with the alpha that you define (these properties are the same as those used in draw_background_ext).

NOTE : Blending is not available for the HTML5 target unless WebGL is enabled.


Example :

draw_background_tiled_ext(bck_Sky, x, y, 2, 2, c_red, 0.5);

This will draw the background indexed in "bck_Sky" at the instances own x and y position, double it's stored size and tiled in every direction in the room, as well as blended with the color red and partially transparent.
