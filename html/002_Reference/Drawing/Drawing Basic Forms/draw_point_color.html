draw_point_color

Draws a single pixel in a given color.

Syntax :

draw_point_color(x, y, col1);


ArgumentDescription
x The x coordinate of the point.
y The y coordinate of the point.
col1 The color of the point.


Returns : N/A


Description

With this function you can draw a single pixel anywhere on the screen with a color that you define. The color settings will over-ride the base color set with the function draw_set_color.


Example :

draw_point_color(50, 50, c_red);

This would draw a red pixel at (50,50).
