current_month

Returns the current month.

Syntax :

current_month;


Returns : Real


Description

This read only variable returns the curremnt month as a numeric value where 1 is January and 12 is December.


Example :

draw_text((32, 32, "Today is " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +".");

The above code will draw the day, month and year.
