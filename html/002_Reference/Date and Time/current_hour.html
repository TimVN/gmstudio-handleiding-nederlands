current_hour

Returns the hour corresponding to the current moment.

Syntax :

current_hour;


Returns : Real


Description

This read only variable will return the hour that corresponds to the current moment using Universal Time (ie: the hour returned ignores Daylight Saving Time).


Example :

draw_text(32, 32, "The time is " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second));

The above code would draw the current international time on the screen.
