Date And Time

This section deals with functions related to dates and time.

GameMaker:Studio has a great many functions and variables related to getting the date and time and calculating other values based off of them.

Time Variables

The following built in variables can be used to get specific date and time components:

  1. current_time
  2. current_second
  3. current_minute
  4. current_hour
  5. current_day
  6. current_weekday
  7. current_month
  8. current_year
