current_time

Returns the milliseconds passed since the game was started.

Syntax :

current_time;


Returns : Real


Description

This read only variable will return the number of milliseconds that have passed since the current game was started.


Example :

if current_time > 600000
   {
   msg = show_question_async("Would you like to rate?");
   }

The above code checks to see if more than 10 minutes have passed before asking the user a question.
