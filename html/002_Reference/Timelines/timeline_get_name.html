timeline_get_name

Gets the name of a given time line.

Syntax :

timeline_get_name( ind );


ArgumentDescription
ind The index of the time line to check the name of.


Returns : String


Description

This function can be used to get the name of a time line as a string. if the time line has been created dynamically using the timeline_add function, the name returned will have the format "__newtimelineN" where "N" is the number of the time line (starting from 0).


Example :

str = timeline_get_name(global.tl);

The above code will store the name of the time line indexed in the variable "global.tl".
