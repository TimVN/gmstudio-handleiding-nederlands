achievement_login

Log the user into the leaderboards page for the chosen platform.

Syntax :

achievement_login()


Returns : N/A


Description

This function logs the user into appropriate leaderboard and achievement service. If the service is not avaialable, the user is logged into a "pretend" game center and all achievements and scores are stored on the device so that when the actual service is available, these details can be uploaded.

Example :

achievement_login();

The above code will log you into the choosen leaderboard and achievement service.
