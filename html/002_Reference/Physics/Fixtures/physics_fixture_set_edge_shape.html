physics_fixture_set_edge_shape

This function sets the shape of the fixture to that of a circle with the specified radius.

Syntax :

physics_fixture_set_edge_shape(fixture, x1, y1, x2, y2)


ArgumentDescription
fixture the index of the fixture
x1 start x position for the edge
y1 start y position for the edge
x2 end x position for the edge
y2 end x position for the edge


Returns : N/A


Description

This function defines an "edge" fixture shape. An edge shape is simply a line that will generate a collision when other fixtures over lap it, and can be very useful for generating (for example) terrain, or for creating borders around a room.


Example :

var xx = 0;
var y1 = room_height - 100;
var y2 = room_height - 50 - irandom(100);
for (var i = 0; i < 10; i++;)
   {
   var fix = physics_fixture_create();
   physics_fixture_set_edge_shape(fix, xx, y1, xx + 50, y2);
   physics_fixture_bind(fix);
   physics_fixture_delete(fix);
   xx += 50;
   y1 = y2;
   y2 = room_height - 50 - irandom(100);
   }

The above code will create a line of "edge" fixtures with a variety of heights over the length of the room.
