window_handle

Returns the id of the game window.

Syntax :

window_handle();


Returns: Real


Description

With this function you can get the internal Windows id (handle). This function is really only useful for extension writers who need the window_handle() to call Win32 API's in DLL code.

NOTE : This functions is for Windows only.


Example :

win_id = windows_handle();

The above code will store the game window id in a variable.
