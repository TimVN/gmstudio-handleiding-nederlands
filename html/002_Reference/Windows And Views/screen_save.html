screen_save

Saves a screenshot of the game as a *.png.

Syntax:

screen_save(fname);


ArgumentDescription
fname The filename for the screenshot to save


Returns : N/A


Description

With this function you can save a screenshot of the game as it currently appears in the game window or browser. This is an image of the whole display region and not just a specific section or view. Should you need more control, you should use the function screen_save_part. The file will be saved to the working directory of the game, which in windows is located at "C:\Users\YOURUSERNAME\AppData\Local\YOURGAMENAME\"

NOTE : This function does not work on the HTML5 target.


Example :

if keyboard_check_pressed(ord("S"))
   {
   screen_save(working_directory + "\Screeens\Screen_"+string(num)+".png")
   num += 1;
   }

The above code takes a screenshot of the game when the user presses the "S" key. This screen shot is saved to a specific directory and numbered using a variable which is then incremented.
