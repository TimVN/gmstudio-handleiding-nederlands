room_set_background_color

Sets the background color of a given room.

Syntax :

room_set_background_color( ind, col, show );


ArgumentDescription
ind The index of the room to set the background color of.
col The new background color.
show Whether to show background color (true) or not (false).


Returns : N/A


Description

With this function you can set the background color for a room (except the current one) and choose whether it should be drawn (true>/tt>) or not (false).


Example :

global.rm = room_add();
room_set_background_color(global.rm, c_maroon, 1);

The above code will add a new room to the game and then set the background color for that room to marron and make the color visible too.
