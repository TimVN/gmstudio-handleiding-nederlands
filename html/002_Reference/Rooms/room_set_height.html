room_set_height

Sets the height, in pixels, of a given room.

Syntax :

room_set_height( ind, h );


ArgumentDescription
ind The index of the room to set the height of.
h The new height of the room in pixels.


Returns : N/A


Description

With this function you can change (or set) the height of any room in your game except the current one.


Example :

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_caption(global.myroom, 'Game Room');
room_set_persistent(global.myroom, false);

This will create a new room and store it's index in the variable "global.myroom". It will then set its width to 640 pixels, its height to 480 pixels, its caption to 'Game Room' and its persistence to 'false'.
