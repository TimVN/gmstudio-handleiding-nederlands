room_caption

Holds the room caption as a string.

Syntax :

room_caption;


Returns: String


Description

When you create a room, you have the option of giving it a "caption" in the Room Editor. By setting this variable to a string you are changing the room caption that was defined, or you can read this variable to get the caption as it was defined. If no caption was created, or you wish to set no caption, use an empty string "".


Example :

if room_caption="" room_caption = room_get_name(room);

The above code will check to see if the room has a caption and if it does not it sets the caption to be the same as the name of the room.
