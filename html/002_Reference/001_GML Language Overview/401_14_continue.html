The "Continue" Statement

This section covers the "continue" statement.

The continue statement has the form :

continue


If used inside of a statement that forms a loop (repeat, while, do... until or for), it will immediately jump back to the beginning of the loop as if the loop had run through and looped back (if used outside of a loop it acts as an exit statement, exiting the event). Below is an example of it's use in a "for" loop :

{
var i;
for (i = 0; i < 10; i += 1)
   {
   if array[i] = "" continue;
   array[i] = "";
   }
}


The above code will skip back to the beginning of the loop if the array[i] value is already an empty string.
