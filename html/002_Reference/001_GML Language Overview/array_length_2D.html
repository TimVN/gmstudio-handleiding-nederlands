array_length_2D

Returns the length of a given dimension of a 2D array.

Syntax :

array_length_2D(array, n);


ArgumentDescription
array The array to check.
n The entry of the array to get the length of.


Returns: Real


Description

With this function you can get the length (number of entries) of a the second dimension of an array. You supply the entry number for the first dimension and the function will return the number of second dimension entries that the array has (to find the length of the first dimension use the function array_length_1D). The function will return 0 if the variable given is not an array or is a 1D array.


Example :

for (var i = array_length_1D(a); i > -1; i--;)
   {
   for (var j = array_length_2D(a, i); j > -1; i--;)
      {
      a[i, j] = -1;
      }
   }

The above code will loop through a 2D array and set each entry to -1.
