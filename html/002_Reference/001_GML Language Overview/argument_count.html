argument_count

Returns the number of arguments passed to a script.

Syntax :

argument_count;


Returns : Real


Description

This read-only variable returns the number of "arguments" that are passed through to a script.


Example :

var i, arg;
for (i = 0; i < 5; i += 1;)
   {
   if argument_count > i
      {
      arg[i] = argument[i]
      }
   else
      {
      arg[i] = -1;
      }
   }

The above code uses the argument_count variable to initialise variables, and is an example of how to permit a script to accept a variable number of arguments.
