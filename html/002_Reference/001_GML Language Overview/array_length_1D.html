array_length_1D

Returns the length of the first dimension of an array.

Syntax :

array_length_1D(array);


ArgumentDescription
array The array to check.


Returns: Real


Description

With this function you can get the length (number of entries) of an array. This array can be either 1D, where the number of entries in the array will be returned, or 2D, in which case the first dimension length will be returned (should you need to find the length of the second dimension you should use the function array_length_2D). The function will return 0 if the variable given is not an array.


Example :

for (var i = array_length_1D(a); a > -1; i--;)
   {
   a[i] = -1;
   }

The above code will loop through an array and set each entry to -1.
